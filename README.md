## Bắt đầu

### Yêu cầu

* [ReaLTaiizor](https://github.com/Taiizor/ReaLTaiizor).

### Cài đặt

* Cài đặt package nuget

```sh
Install-Package ReaLTaiizor
```

* Import cơ sở dữ liệu (Database/Database.sql) vào MSSQL
